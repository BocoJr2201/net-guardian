# Net Guardian - Dashboard com Análise de IP Suspeito

Esta versão mantém a ferramenta atual e adiciona o módulo de Análise de IP Suspeito.

## Funcionalidades

1. Análise de Host/Portas
2. Análise defensiva de IP suspeito
3. Geração de relatórios TXT
4. Dashboard web visual

## Como executar

```bash
pip install flask
python app.py
```

Depois acesse:

```text
http://127.0.0.1:5000
```
